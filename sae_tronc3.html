SAE 2.04

Réalisation

Durant cette SAE, j'ai pu utiliser un capteur température, et avec celui-ci j'ai pu récolter et visualiser les données grâce à l'utilisation de python, grafana et OpenStack.

Résultat

Grâce à cette SAE, j'ai pu voir comment on crée un capteur de température, comment on fait pour récupérer les données (grâce à python) et de pouvoir concrètement voir à quoi sert Grafana.

Les Compétences

  • Création du capteur de mesure de température.
  • Visualiser les données acquises par le capteur.
  • Développement du code python permettant d'acquérir les données.
  • Installation d'infrastructure sur OpenStack.
  • Stockage des données mesuré sur une base de données.
